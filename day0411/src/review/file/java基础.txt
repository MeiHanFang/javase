1、能够说出java语言的跨平台(可移植)特性
.java文件通过编译生成.class文件，再通过类加载器加载到JVM虚拟机执行，不同的平台有不同的虚拟机，因此java语言是跨平台的。

2、说出jvm\jre\jdk分别指的什么
jvm java虚拟机
jre java运行时环境
jdk java开发工具包

3、能够说出为什么配置环境变量
为了让jdk的bin目录下的所有命令可以在任意目录下使用

3、能够说出break和continue两个关键字的作用
break      跳出循环 结束循环 应用于循环语句和switch语句
continue   跳出本次循环 继续下一次循环 应用于循环语句
